# HeptaCore Brand Foundation

## Name

HeptaCore

## Tagline

Inteligencia multidimensional + gestion de RRSS

## Direction

Premium minimalism: white, black, graphite, fog, bone, and a restrained deep teal accent.

The core visual signal is an abstract circular mark: ink, orbit, signal, nucleus. The mark should feel original and strategic, not like a generic AI symbol.

## Tokens

```css
--hc-ink: #0A0A0A;
--hc-graphite: #2B2B2B;
--hc-fog: #8E8E93;
--hc-bone: #F6F6F7;
--hc-abyss: #0F2E33;
--hc-white: #FFFFFF;
```
